# ENS NameX

An ENS domain exchange.

## Key Files

* `contracts/DirectListing.sol` - the exchange contract
* `src/` - UI using Truffle's Drizzle
* `test/DirectListing.test.js` - example interacting with contract

## Running

With an already running local node:

1. `truffle migrate`
2. `yarn cp-contracts`
3. `yarn start`
